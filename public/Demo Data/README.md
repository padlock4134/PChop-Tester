# Demo Data Files for PorkChop Ed Tech Reports

This folder contains mock data files for demonstration purposes for the 10 culinary education reports.

## Report Categories

### 1. Student Progress
- `skill-mastery-tracking.*` - Student progress in culinary skills and certifications
- `learning-analytics.*` - Engagement, quiz scores, and knowledge retention

### 2. Class Analytics  
- `class-performance.*` - Class scores, completion rates, and knowledge gaps
- `live-session-metrics.*` - Attendance and engagement in live cooking sessions

### 3. Culinary Metrics
- `recipe-performance.*` - Recipe success rates and cooking outcomes  
- `technique-analysis.*` - Progress on specific culinary techniques and skills

### 4. Operations
- `kitchen-management.*` - Equipment usage and operational efficiency
- `safety-compliance.*` - Food safety violations and compliance metrics

### 5. Engagement
- `platform-usage.*` - Student interaction with the learning platform
- `community-engagement.*` - Social learning and community participation

## File Formats

Each report is available in 3 formats:
- **CSV** - Raw data for analysis and custom reporting
- **PDF** - Formatted report with charts and summaries  
- **Excel** - Spreadsheet with formulas and pivot tables

## Usage

These files are used by the Reports modal in the Profile component to demonstrate the download functionality for culinary school administrators and instructors.

**Note**: These are demo files with mock data for presentation purposes only.
